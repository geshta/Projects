MILK DELIVERY MANAGEMENT SYSTEM
================================

Version : 1.0
Platform: Windows (Desktop Application)

--------------------------------------------------
ABOUT THE APPLICATION
--------------------------------------------------
The Milk Delivery Management System is a desktop-based application designed to
help dairy businesses manage customers, daily milk entries, billing, and reports
in a simple and efficient way. The application reduces manual work by automating
monthly bill calculations, report generation, and customer communication.

--------------------------------------------------
FEATURES
--------------------------------------------------
• Customer management (Add, Edit, Delete customers)
• Daily milk entry tracking
• Automatic monthly and yearly report generation
• Excel-based data storage for easy backup
• WhatsApp bill message generation
• Search and filter customers
• Easy-to-use graphical interface
• Data organized month-wise and year-wise

--------------------------------------------------
SYSTEM REQUIREMENTS
--------------------------------------------------
• Operating System: Windows 7 / 8 / 10 / 11 (64-bit)
• Minimum RAM: 4 GB
• Disk Space: 100 MB free space
• Internet: Required only for WhatsApp messaging

(No Python installation required when using the installer)

--------------------------------------------------
INSTALLATION
--------------------------------------------------
1. Run the installer file (MilkDeliveryApp_Setup.exe)
2. Follow on-screen instructions
3. After installation, launch the application from Desktop or Start Menu

--------------------------------------------------
FIRST TIME USAGE
--------------------------------------------------
• Enter your business details when prompted
• Add customer details before entering milk data
• Select the correct month before making entries
• Always SAVE the Excel sheet after entering daily data

--------------------------------------------------
IMPORTANT NOTES
--------------------------------------------------
• Always save the monthly Excel sheet after entering data
• Do not delete files inside the "data" folder manually
• Backup the "data" folder regularly to avoid data loss
• Ensure correct phone numbers for WhatsApp messages

--------------------------------------------------
DATA STORAGE
--------------------------------------------------
All data is stored locally inside the application folder:
• Customer data
• Monthly milk records
• Reports and status files

You can back up the entire "data" folder safely.

--------------------------------------------------
TROUBLESHOOTING
--------------------------------------------------
• If reports show 0 values, ensure the monthly sheet is saved
• If WhatsApp does not open, check internet connection
• Restart the application if any unexpected error occurs
